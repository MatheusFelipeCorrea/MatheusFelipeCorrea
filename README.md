<h3>🚀 Aplicações</h3>

<h3> 🟢 <a href="https://github.com/ICEI-PUC-Minas-PMGES-TI/pmg-es-2024-2-ti2-3687100-atenasbuffet" target="_blank">AthenasBuffet</a></h3>
<p>
Sistema web desenvolvido para modernizar e otimizar a gestão de buffets que enfrentam desorganização com métodos manuais ou limitados (como WhatsApp). A plataforma integra funcionalidades como:
</p>
<ul>
  <li>Solicitação de orçamento</li>
  <li>Personalização de cardápios</li>
  <li>Agendamento de eventos</li>
  <li>Seleção de garçons</li>
  <li>Coleta de feedback</li>
</ul>
<p>
Foco em eficiência, centralização de informações e personalização, elevando a experiência do cliente e a competitividade no mercado.
</p>

<h3> 🟢 <a href="https://github.com/MatheusFelipeCorrea/BudgetBuddy" target="_blank">BudgetBuddy</a></h3>
<p>
Aplicação web para gestão financeira pessoal. Utiliza <strong>Supabase</strong> para armazenamento de dados e oferece funcionalidades como:
</p>
<ul>
  <li>Acompanhamento de saldo</li>
  <li>Controle de vencimentos futuros</li>
  <li>Classificação de gastos por categoria</li>
  <li>Visualização de gráficos financeiros mensais</li>
  <li>Dicas financeiras e criação de metas</li>
  <li>Painel de extrato personalizado</li>
</ul>

<h3> 🟢 <a href="https://github.com/pucmg-aulas/turmamanha-g3-puc-lovers" target="_blank">Java Parkings</a></h3>
<p>
Sistema de controle para estacionamentos em grandes centros urbanos. Desenvolvido pela Xulambs Inc., tem como objetivo otimizar:
</p>
<ul>
  <li>Ocupação e reserva de vagas</li>
  <li>Cobranças automatizadas</li>
  <li>Eficiência na gestão de espaços</li>
</ul>
<p>
Melhora a experiência do usuário ao lidar com a escassez de vagas.
</p>
